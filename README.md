# Учёт домашних расходов

### Описание
Простое приложение для учёта расходов с использованием SQLite и tkinter.

### Структура БД
Таблица `expenses`:
- `id` — уникальный номер;
- `title` — название расхода;
- `amount` — сумма;
- `date` — дата.

### Функции
- Добавление расходов;
- Просмотр списка;
- Удаление записей.
